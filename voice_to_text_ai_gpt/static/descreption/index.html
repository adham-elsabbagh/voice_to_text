Installation Instructions

  1. Download and extract the module to your Odoo addons directory
  2. Install required Python dependencies:
    pip install openai openai-whisper
  3. Update your module list in Odoo
  4. Install the "Voice to Text" module from Apps
  5. Configure OpenAI API key in System Parameters:
    Key: openai.api_key
    Value: your-openai-api-key
  6. Grant microphone permissions in your browser
  7. Start using voice recording in CRM leads and activities!
